# 📈 Equity Research Terminal

## End-to-End QA Engineering & Equity Research Web Application

Equity Research Terminal is a web-based application developed using Python and Streamlit to simplify stock research by providing company financials, valuation models, forensic analysis, annual report analysis, news, watchlists, and portfolio management in a single platform.

This repository is also used as an end-to-end Quality Assurance project to demonstrate the complete Software Testing Life Cycle (STLC), including requirement analysis, manual testing, API testing, SQL validation, and Playwright automation.

---

## Project Overview

The application helps investors, students, and equity research analysts perform company research without switching between multiple financial websites.
